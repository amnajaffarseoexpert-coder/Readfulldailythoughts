# Free Thought Daily Design Rules

Use this file as the source of truth for every future page, template, block, and component design in this WordPress site.

## Brand Identity

- Brand name: Free Thought Daily / FTD.
- Voice: independent, serious, editorial, global, thoughtful.
- Core line: "Independent Minds. Critical Thinking. A Better World."
- The site should feel like a premium international news publication, not a generic WordPress magazine theme.

## Color System

- Primary navy: `#061120`
- Deep header navy: `#020c1a`
- Brand red: `#d20919`
- Paper background: `#fbfaf7`
- Soft panel background: `#f4f0eb`
- Body text: `#061120`
- Muted text: `#5f6874`
- Border line: `#e7e0d8`
- Gold accent, only for tiny premium details: `#d3a15f`

Do not use bright royal blue as a main brand color. Avoid generic Newsup blues such as `#1558d6`, `#1e73be`, and similar saturated blue navigation/widget colors.

## Typography

- Editorial headlines should use a serif face such as Georgia or Times New Roman.
- Navigation, labels, buttons, metadata, and utility UI should use a clean sans-serif such as Arial or Helvetica.
- Headlines should feel newspaper-like: confident, compact, and high contrast.
- Do not use overly playful, rounded, or tech-dashboard typography.

## Layout Rules

- Header must match the FTD identity: dark top strip, white main nav, FTD logo, red subscribe button, and restrained section links.
- Home, archive, category, search, and single-post pages must share the same brand header treatment.
- Section headings should be uppercase, dark navy, with a small red accent line or red arrow.
- Cards should be editorial and compact. Use 4px border radius unless there is a strong reason otherwise.
- Use paper and soft off-white surfaces instead of pure gray content blocks.
- Avoid nested cards and heavy blue widget boxes.

## Imagery

- Prefer real editorial/news imagery with strong subject matter.
- The homepage hero uses the earth image style: global, cinematic, dark on the right, light editorial copy space on the left.
- Do not use abstract blue technology backgrounds for generic page headers unless the page topic specifically requires it.
- Logo should appear as the full FTD mark when there is enough space, not only the cropped globe mark.

## Components

- Primary buttons: red background `#d20919`, white text, uppercase, 4px radius.
- Secondary links: dark text with a red underline/accent.
- Breaking-news labels: red background with white uppercase text.
- Sidebar widgets: soft paper panels with navy headings and red accents; never royal-blue title ribbons.
- Search fields: white inputs with subtle borders and red or navy action button.

## Page Coverage

Apply these rules to:

- Homepage/front page
- Category archives
- Tag archives
- Search results
- Single posts
- Static pages
- Sidebar widgets
- Header/navigation
- Footer

Any new design work should first check this file, then inspect existing templates/CSS before editing.
